# VICI Limited - Frontend

Welcome to the frontend repository for VICI Limited! This guide will help you set up the project locally so you can connect and test the backend API.

## Tech Stack

- **Framework:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4
- **Routing:** React Router DOM
- **Icons:** React Icons

---

## 🚀 Getting Started

### 1. Prerequisites

Ensure you have **Node.js** installed on your machine. You can download it from [nodejs.org](https://nodejs.org/).

### 2. Clone the Repository

Open your terminal and clone the project to your local machine:

```bash
git clone <repository-url>
```

## Install Dependencies

Navigate into the project folder and install the required NPM packages:

cd vici_ltd
npm install

## Start the Development Server

Run the following command to spin up the local Vite server:

    Bash
    npm run dev

## Project Routes

When the server is running, you can access the following pages:

## Public Routes

/ - VICI Limited Corporate Homepage (The Umbrella Brand)

/farms - VICI Farms Division (Detailed view for mushrooms, chili, livestock)

/beverages - VICI Beverages Division (Placeholder launch page)

## Hidden Admin Routes

/admin-login - Secure Login Portal (For entering staff/admin credentials)

/dashboard - Staff Dashboard (For recording sales/inventory)

/admin-dashboard - Admin Dashboard (For role management and financial overview)
